# Web Image Converter (Batch Processing)

Это одностраничное клиентское веб-приложение (SPA) для пакетной конвертации изображений в браузере. Приложение работает полностью на клиенте, не отправляя файлы на сервер, что делает его безопасным и конфиденциальным.


## Основные возможности
* **Пакетная обработка:** Возможность загружать множество изображений за раз, просматривать их статус в очереди и конвертировать их параллельно (до 6 потоков).
* **Экспорт в ZIP:** Скачивание всех успешно конвертированных изображений одним ZIP-архивом.
* **Форматы:** Конвертация в JPG, JPEG, PNG и WebP.
* **Качество:** Настройка степени сжатия для JPEG и WebP (сразу для всей пачки).
* **Изменение размера (Ресайз):** Автоматическое пропорциональное уменьшение ширины/высоты изображений до заданного лимита (800px, 1200px, 1920px, 2560px).
* **Суммарная статистика:** Расчет общего веса файлов до и после обработки с показом общего процента экономии места.
* **Поиск и фильтрация:** Поиск файлов в очереди по названию и фильтрация по статусу обработки (Все, Ожидают, В процессе, Готово, Ошибки).
* **Прозрачность:** Автоматическая замена прозрачного фона на белый при конвертации в JPEG/JPG.
* **Индивидуальная статистика:** Сравнение размера "До -> После" и процент сжатия для каждого изображения.
* **Drag & Drop / Вставка (Ctrl+V):** Удобная зона загрузки перетаскиванием, добавление файлов в процессе и поддержка вставки скриншотов из буфера обмена.

## Как запустить проект локально

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте в браузере адрес:
   **[http://localhost:5173/](http://localhost:5173/)**

## Сборка для публикации (GitHub Pages)

Для сборки оптимизированной версии проекта выполните:
```bash
npm run build
```

Собранные файлы появятся в директории `dist/`. Пути к ресурсам настроены как относительные (`base: './'`), поэтому проект готов к публикации на GitHub Pages.
